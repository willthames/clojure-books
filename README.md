# books

A proof of concept of XML parsing in Clojure.
Parses books.xml and prints the titles of all books by Eva Corets

## Usage

```
lein deps
lein compile
lein run
```
prints
```
([Maeve Ascendant] [Oberon's Legacy] [The Sundered Grail])
```
